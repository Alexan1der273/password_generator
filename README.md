# password_generator
Простой генератор паролей на Python с tkinter

# 🔐 Генератор паролей v0.0.2

Улучшенный генератор надёжных случайных паролей с графическим интерфейсом на Python (tkinter).

## Возможности

- **Ползунок и поле ввода** для выбора длины пароля (4–32 символа)
- **Выбор типов символов**:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Спецсимволы (!@#$%^&*...)
- **Индикатор сложности** пароля (от "очень слабый" до "очень сильный") на основе энтропии
- **Гарантированное включение** хотя бы одного символа из каждого выбранного типа
- **Показ/скрытие пароля** (кнопка 👁/🙈)
- **Копирование в буфер обмена** одним кликом
- **Кнопка "Очистить"** для сброса поля с паролем
- **Статусная строка** с уведомлениями
- **Современный дизайн** с цветовой индикацией

## Установка и запуск

1. Убедитесь, что установлен Python 3.14 или выше.

2. Установите библиотеку `pyperclip` для копирования в буфер обмена:
   
   - **Windows**:
     ```bash
     pip install pyperclip
     ```
   - **Linux / macOS**:
     ```bash
     pip3 install pyperclip
     ```
3. Запустите программу:
   
   - **Windows**:
     ```bash
     python password_generator.py
     ```
   - **Linux / macOS**:
     ```bash
     python3 password_generator.py
     ```

## Использование

1. Выберите длину пароля ползунком или введите в поле (от 4 до 32).
2. Отметьте нужные типы символов (обязательно хотя бы один).
3. Нажмите **"Сгенерировать пароль"**.
4. При необходимости нажмите на иконку глазика 👁, чтобы увидеть пароль.
5. Нажмите **"Копировать в буфер"**.
6. Используйте кнопку **"Очистить"** для сброса поля с паролем.
