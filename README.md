# password_generator
Простой генератор паролей на Python с tkinter

# 🔐 Генератор паролей v0.0.1

Простой генератор надёжных случайных паролей с графическим интерфейсом на Python (tkinter).

## Возможности

- Выбор длины пароля (от 1 символа)
- Выбор типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Спецсимволы (!@#$%^&*...)
- Гарантированное включение **хотя бы одного символа** из каждого выбранного типа
- Копирование пароля в буфер обмена одним кликом
- Простой и интуитивно понятный интерфейс

## Установка и запуск

1. Убедитесь, что у вас установлен Python версии 3.6 или выше.

2. Установите библиотеку `pyperclip` для копирования в буфер обмена:
   - **Windows**:
     ```bash
     pip install pyperclip
     ```
   - **Linux / macOS**:
     ```bash
     pip3 install pyperclip
     ```
3. Запустите программу:
   - **Windows**:
     ```bash
     python password_generator.py
     ```
   - **Linux / macOS**:
     ```bash
     python3 password_generator.py
     ```

## Использование

1. В поле "Длина пароля" введите желаемое количество символов (по умолчанию 12).
2. Отметьте необходимые типы символов (заглавные, строчные, цифры, спецсимволы).
3. Нажмите кнопку **"Сгенерировать пароль"**.
4. Нажмите **"Копировать в буфер"**, чтобы скопировать пароль.

## Пример

```
Длина: 12
Символы: Заглавные + Строчные + Цифры + Спецсимволы
Сгенерированный пароль: aB3$kL9#mN2x
```

## Требования

- Python 3.x
- pyperclip (устанавливается отдельно)
